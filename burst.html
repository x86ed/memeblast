HTML5 Canvas

This tool creates its images using the <canvas> element from HTML 5. Unfortunately, your browser currently doesn't support this latest version of HTML.

You can upgrade your browser to a modern browser such as the latest versions of Firefox, Google Chrome, Safari, or Opera. Because of the way Microsoft responds to emerging standards, Internet Explorer is usually a couple of years behind these others. Canvas support is included in Internet Explorer 9.

Please note that any images you create with this tool will work on all browsers, including those without HTML5 support.

Documentation

This image generator produces a two-tone starburst effect that is a popular background for page elements that need to draw the eye.

Fill out the form above, click 'create' then right click the blue button to download your image.

The dimenions of the image are in pixels, and the burst origin is relative to the top left of the image (increasing down and to the right). If you want the origin to be on the image, you can click on the image above. You can set values outside the image bounds using the form.

The color specifications can be anything that CSS can understand, so you can use X11 color names (e.g. white or goldenrod); 3 digit or 6 digit hex colors, prefixed by a hash (e.g. #369 or #f9f8f7); and rgb or rgba specifications (e.g. rgb(255,0,0) or rgba(255,0,0,0.4)). Note that when using rgb and rgba values, the first three parameters are integers from 0-255, while for rgba the last parameter is an opacity from 0-1.

The sunburst color is a gradient from the inside of the burst to the second nearest corner. This may seem an odd or arbitrary choice, but is probably what you actually need. If it isn't, then I'm sorry. There are so many possible combinations, you may have to generate your own image.
